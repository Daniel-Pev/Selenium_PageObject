<h2>Автотесты на UI проекта «Битва покемонов»</h2>

> **Статус проекта:**
> Проект закрытый для POST запросов, но GET можно выполнять без токена: https://pokemonbattle.ru/
> 


## Описание проекта и задачи
Автоматизировать часть проверок регресса с помощью Pytest и Selenium

## Детали реализации

1. Автотесты написаны с применением PyTest
2. Используется библиотека Selenium
3. Параметризированные тесты с использованием декоратора
4. Параллельный запуск тестов через pytest-xdist
5. Создание отчетов в Allure reports
6. Фреймворк написан с ипользованием паттерна Page Object Model 

## Локальный запуск тестов (из терминала)
1. Скачать проект
2. Перейти через терминал в директорию проекта
3. Выполнить команду

Создаём виртуальное окружение внутри папки проекта.

``` markdown
python -m venv venv
```

``` markdown
venv\Scripts\activate
```

4. Устанавливаем библиотеки

``` markdown
pip install -r requests
```

Подправить конфигурационный файл для тестов 
``` markdown
common/conf.py
```

Запускаем
``` markdown
pytest 
```
